# DC-ARK Maker

##### Setup

* ALL:

    * python3 is required

* MacOS:
    
    * `brew install openssl@1.1`

* Ubuntu:
    * https://packages.ubuntu.com/focal/libssl1.1

##### Note: Working on Linux and Windows, MacOS is experimental

###### Need to run as root/admin

* `pip install -r requirements.txt`

* `./main.py`

* choose your drive wisely....
